# Kora Java JDK Scheduling

Пример модуля JDK Scheduling в Kora.

В примере использовались модули:
- [Scheduling JDK](https://kora-projects.github.io/kora-docs/ru/documentation/scheduling/#_1)

## Build

Собрать артефакт:

```shell
./gradlew shadowJar
```

## Run

Запустить локально:
```shell
./gradlew run
```

## Test

Протестировать локально:
```shell
./gradlew test
```
