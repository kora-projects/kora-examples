# Kora Java Guides

Kora Java guide applications are runnable companion projects for the guides in the Kora documentation. Each entry is a Gradle module that can be compiled, tested, and used as the source reference for the matching guide.

## Common Commands

Use the Gradle path shown in a module README, for example:

```shell
./gradlew :guides:java:kora-java-guide-cache-app:classes
./gradlew :guides:java:kora-java-guide-cache-app:testClasses
./gradlew :guides:java:kora-java-guide-cache-app:test
```
