# Kora GraalVM examples

Kora GraalVM examples show Java services configured for native image builds with Kora.

| Example | Description | Kora modules |
| --- | --- | --- |
| [`kora-java-graalvm-crud-cassandra`](kora-java-graalvm-crud-cassandra) | Native CRUD HTTP service backed by Cassandra and Redis cache. | [CassandraDatabaseModule](https://kora-projects.github.io/kora-docs/ru/documentation/database-cassandra/), [RedisCacheModule](https://kora-projects.github.io/kora-docs/ru/documentation/cache/), [UndertowHttpServerModule](https://kora-projects.github.io/kora-docs/ru/documentation/http-server/), [ResilientModule](https://kora-projects.github.io/kora-docs/ru/documentation/resilient/), [ValidationModule](https://kora-projects.github.io/kora-docs/ru/documentation/validation/), [JsonModule](https://kora-projects.github.io/kora-docs/ru/documentation/json/), [MetricsModule](https://kora-projects.github.io/kora-docs/ru/documentation/metrics/), [OpenApiManagementModule](https://kora-projects.github.io/kora-docs/ru/documentation/openapi-management/) |
| [`kora-java-graalvm-crud-jdbc`](kora-java-graalvm-crud-jdbc) | Native CRUD HTTP service backed by JDBC and Caffeine cache. | [JdbcDatabaseModule](https://kora-projects.github.io/kora-docs/ru/documentation/database-jdbc/), [CaffeineCacheModule](https://kora-projects.github.io/kora-docs/ru/documentation/cache/), [UndertowHttpServerModule](https://kora-projects.github.io/kora-docs/ru/documentation/http-server/), [ResilientModule](https://kora-projects.github.io/kora-docs/ru/documentation/resilient/), [ValidationModule](https://kora-projects.github.io/kora-docs/ru/documentation/validation/), [JsonModule](https://kora-projects.github.io/kora-docs/ru/documentation/json/), [MetricsModule](https://kora-projects.github.io/kora-docs/ru/documentation/metrics/), [OpenApiManagementModule](https://kora-projects.github.io/kora-docs/ru/documentation/openapi-management/) |
| [`kora-java-graalvm-crud-r2dbc`](kora-java-graalvm-crud-r2dbc) | Native CRUD HTTP service backed by R2DBC and Caffeine cache. | [R2dbcDatabaseModule](https://kora-projects.github.io/kora-docs/ru/documentation/database-r2dbc/), [CaffeineCacheModule](https://kora-projects.github.io/kora-docs/ru/documentation/cache/), [UndertowHttpServerModule](https://kora-projects.github.io/kora-docs/ru/documentation/http-server/), [ResilientModule](https://kora-projects.github.io/kora-docs/ru/documentation/resilient/), [ValidationModule](https://kora-projects.github.io/kora-docs/ru/documentation/validation/), [JsonModule](https://kora-projects.github.io/kora-docs/ru/documentation/json/), [MetricsModule](https://kora-projects.github.io/kora-docs/ru/documentation/metrics/), [OpenApiManagementModule](https://kora-projects.github.io/kora-docs/ru/documentation/openapi-management/) |
