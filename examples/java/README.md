# Kora Java examples

Kora Java examples are independent Gradle applications that demonstrate Kora modules in small, focused services.

| Example | Description | Kora modules |
| --- | --- | --- |
| [`kora-java-cache-caffeine`](kora-java-cache-caffeine) | In-memory cache example based on Caffeine. | [CaffeineCacheModule](https://kora-projects.github.io/kora-docs/ru/documentation/cache/) |
| [`kora-java-cache-redis`](kora-java-cache-redis) | Redis-backed cache example. | [RedisCacheModule](https://kora-projects.github.io/kora-docs/ru/documentation/cache/) |
| [`kora-java-camunda-engine`](kora-java-camunda-engine) | Camunda 7 BPMN engine with REST API. | [CamundaEngineBpmnModule](https://kora-projects.github.io/kora-docs/ru/documentation/camunda7-bpmn/), [CamundaRestUndertowModule](https://kora-projects.github.io/kora-docs/ru/documentation/camunda7-rest/) |
| [`kora-java-camunda-zeebe-worker`](kora-java-camunda-zeebe-worker) | Camunda 8 Zeebe worker example. | [ZeebeWorkerModule](https://kora-projects.github.io/kora-docs/ru/documentation/camunda8-worker/) |
| [`kora-java-config-hocon`](kora-java-config-hocon) | Reading application configuration from HOCON. | [HoconConfigModule](https://kora-projects.github.io/kora-docs/ru/documentation/config/) |
| [`kora-java-config-yaml`](kora-java-config-yaml) | Reading application configuration from YAML. | [YamlConfigModule](https://kora-projects.github.io/kora-docs/ru/documentation/config/) |
| [`kora-java-crud`](kora-java-crud) | Full CRUD HTTP service with persistence, cache, resilience, validation, metrics and OpenAPI management. | [UndertowHttpServerModule](https://kora-projects.github.io/kora-docs/ru/documentation/http-server/), [JdbcDatabaseModule](https://kora-projects.github.io/kora-docs/ru/documentation/database-jdbc/), [CaffeineCacheModule](https://kora-projects.github.io/kora-docs/ru/documentation/cache/), [ResilientModule](https://kora-projects.github.io/kora-docs/ru/documentation/resilient/), [ValidationModule](https://kora-projects.github.io/kora-docs/ru/documentation/validation/), [JsonModule](https://kora-projects.github.io/kora-docs/ru/documentation/json/), [MetricsModule](https://kora-projects.github.io/kora-docs/ru/documentation/metrics/), [OpenApiManagementModule](https://kora-projects.github.io/kora-docs/ru/documentation/openapi-management/) |
| [`kora-java-crud-submodule`](kora-java-crud-submodule) | Same CRUD domain split into application, common, pet API and vet API Gradle submodules. | [UndertowHttpServerModule](https://kora-projects.github.io/kora-docs/ru/documentation/http-server/), [JdbcDatabaseModule](https://kora-projects.github.io/kora-docs/ru/documentation/database-jdbc/), [CaffeineCacheModule](https://kora-projects.github.io/kora-docs/ru/documentation/cache/), [ResilientModule](https://kora-projects.github.io/kora-docs/ru/documentation/resilient/), [ValidationModule](https://kora-projects.github.io/kora-docs/ru/documentation/validation/), [JsonModule](https://kora-projects.github.io/kora-docs/ru/documentation/json/), [MetricsModule](https://kora-projects.github.io/kora-docs/ru/documentation/metrics/), [OpenApiManagementModule](https://kora-projects.github.io/kora-docs/ru/documentation/openapi-management/) |
| [`kora-java-database-cassandra`](kora-java-database-cassandra) | Cassandra repository and mapper examples. | [CassandraDatabaseModule](https://kora-projects.github.io/kora-docs/ru/documentation/database-cassandra/) |
| [`kora-java-database-jdbc`](kora-java-database-jdbc) | JDBC repository and mapper examples. | [JdbcDatabaseModule](https://kora-projects.github.io/kora-docs/ru/documentation/database-jdbc/) |
| [`kora-java-database-r2dbc`](kora-java-database-r2dbc) | R2DBC repository and mapper examples. | [R2dbcDatabaseModule](https://kora-projects.github.io/kora-docs/ru/documentation/database-r2dbc/) |
| [`kora-java-database-vertx`](kora-java-database-vertx) | Vert.x SQL client repository and mapper examples. | [VertxDatabaseModule](https://kora-projects.github.io/kora-docs/ru/documentation/database-vertx/) |
| [`kora-java-grpc-client`](kora-java-grpc-client) | gRPC client example. | [GrpcClientModule](https://kora-projects.github.io/kora-docs/ru/documentation/grpc-client/) |
| [`kora-java-grpc-server`](kora-java-grpc-server) | gRPC server example. | [GrpcServerModule](https://kora-projects.github.io/kora-docs/ru/documentation/grpc-server/) |
| [`kora-java-helloworld`](kora-java-helloworld) | Minimal HTTP JSON service. | [UndertowHttpServerModule](https://kora-projects.github.io/kora-docs/ru/documentation/http-server/), [JsonModule](https://kora-projects.github.io/kora-docs/ru/documentation/json/) |
| [`kora-java-http-client`](kora-java-http-client) | Declarative HTTP client example. | [JdkHttpClientModule](https://kora-projects.github.io/kora-docs/ru/documentation/http-client/) |
| [`kora-java-http-server`](kora-java-http-server) | HTTP server example. | [UndertowHttpServerModule](https://kora-projects.github.io/kora-docs/ru/documentation/http-server/) |
| [`kora-java-kafka`](kora-java-kafka) | Kafka producer and listener example. | [KafkaModule](https://kora-projects.github.io/kora-docs/ru/documentation/kafka/) |
| [`kora-java-openapi-generator-http-client`](kora-java-openapi-generator-http-client) | OpenAPI-generated HTTP client. | [OpenAPI Generator](https://kora-projects.github.io/kora-docs/ru/documentation/openapi-codegen/), [JdkHttpClientModule](https://kora-projects.github.io/kora-docs/ru/documentation/http-client/) |
| [`kora-java-openapi-generator-http-server`](kora-java-openapi-generator-http-server) | OpenAPI-generated HTTP server. | [OpenAPI Generator](https://kora-projects.github.io/kora-docs/ru/documentation/openapi-codegen/), [UndertowHttpServerModule](https://kora-projects.github.io/kora-docs/ru/documentation/http-server/) |
| [`kora-java-resilient`](kora-java-resilient) | Resilience features such as retry, timeout and circuit breaker. | [ResilientModule](https://kora-projects.github.io/kora-docs/ru/documentation/resilient/) |
| [`kora-java-s3-client-aws`](kora-java-s3-client-aws) | S3 client example for AWS SDK integration. | [AwsS3ClientModule](https://kora-projects.github.io/kora-docs/ru/documentation/s3-client/) |
| [`kora-java-s3-client-minio`](kora-java-s3-client-minio) | S3 client example for MinIO integration. | [MinioS3ClientModule](https://kora-projects.github.io/kora-docs/ru/documentation/s3-client/) |
| [`kora-java-scheduling-jdk`](kora-java-scheduling-jdk) | Scheduled tasks using the JDK scheduler. | [SchedulingJdkModule](https://kora-projects.github.io/kora-docs/ru/documentation/scheduling/) |
| [`kora-java-scheduling-quartz`](kora-java-scheduling-quartz) | Scheduled tasks using Quartz. | [QuartzModule](https://kora-projects.github.io/kora-docs/ru/documentation/scheduling/) |
| [`kora-java-soap-client`](kora-java-soap-client) | SOAP client example. | [SoapClientModule](https://kora-projects.github.io/kora-docs/ru/documentation/soap-client/) |
| [`kora-java-telemetry`](kora-java-telemetry) | Metrics and OpenTelemetry tracing exporter example. | [MetricsModule](https://kora-projects.github.io/kora-docs/ru/documentation/metrics/), [OpentelemetryGrpcExporterModule](https://kora-projects.github.io/kora-docs/ru/documentation/tracing/) |
| [`kora-java-validation`](kora-java-validation) | Declarative validation examples. | [ValidatorModule](https://kora-projects.github.io/kora-docs/ru/documentation/validation/) |
